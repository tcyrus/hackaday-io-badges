hackaday-io-badges

Description

Based on the Hackaday.io Skull Counter. This nodejs app generates Project Badges for any package on hackaday.io.

Link

To embed a badge in Markdown: 

[![](https://hackaday-io-badges.herokuapp.com/hackaday/{id}.svg)]({url})

To embed a badge in HTML: 

<a href="{url}"><img src="https://hackaday-io-badges.herokuapp.com/hackaday/{id}.svg"></a>

Make me one

This is not affiliated with the Hackaday.
